# The Abacus Accounting — desktop downloads

**Web app:** [accounting.theabacus.org](https://accounting.theabacus.org)

**Desktop installers (this folder):** the published `.dmg` (macOS, Apple silicon), `.msi` (Windows x64), and `.AppImage` (Linux x64) for The Abacus Accounting, with the app version in each filename (e.g. `*4.0.1*`). All files sit **in** `assets/accounting/` — we do not use a `v4.0.1/` (or other version) subfolder.

**Metadata:** see [`../../release-assets.json`](../../release-assets.json) in the repository root.

**Releases:** [The-Abacus-App-Download — Releases](https://github.com/The-Abacus-Foundation/The-Abacus-App-Download/releases) (e.g. tag `v4.0.1` for Accounting).

**Repository root:** [The-Abacus-App-Download](https://github.com/The-Abacus-Foundation/The-Abacus-App-Download).
