# The Abacus PDF — desktop downloads

**Web app:** [pdf.theabacus.org](https://pdf.theabacus.org) — merge, split, compress, and convert PDFs in the browser.

**Desktop installers (this folder):** the latest published `.dmg` (macOS, Apple silicon), `.msi` (Windows x64), and `.AppImage` (Linux x64) for The Abacus PDF, with the app version in each filename (e.g. `*0.4.0*`). All files live **directly in** `assets/pdf/` next to this `README` and `release-assets.json` — we do not use a `latest/` or `v0.4.0/` subfolder.

**Metadata:** see [`release-assets.json`](release-assets.json).

**Releases:** [The-Abacus-App-Download — Releases](https://github.com/The-Abacus-Foundation/The-Abacus-App-Download/releases) (tag the PDF app as published, e.g. `v0.4.0`).

**Repository root:** [The-Abacus-App-Download](https://github.com/The-Abacus-Foundation/The-Abacus-App-Download).
